I wanted to make a retexure of Grandpa's shed in Stardew Valley Expanded, so here it is! 

I'm heavily inspired by the talented elle/junimods (https://www.nexusmods.com/profile/junimods). This mod is designed to look good alongside her building mods. 
Please note that this mod will require SMAPI (https://smapi.io) and Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915) to work.
